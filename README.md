

# 📊 Data Professional Survey Dashboard

This project presents an interactive Power BI dashboard analyzing survey data from data professionals across different roles, countries, and experience levels.

---

## 🔍 Key Insights

- 💰 **Highest Paying Role:** Data Scientist (~94 avg salary)
- 👨‍💻 **Most Popular Language:** Python (dominant across all roles)
- 🌍 **Top Countries:** USA, Canada, India, UK
- 😊 **Work-Life Balance Rating:** 5.74 / 10
- 💵 **Salary Satisfaction:** 4.27 / 10
- 🎂 **Average Age:** 29.87 years

---

## 📌 Dashboard Features

- Job title vs salary comparison  
- Country-wise distribution of survey takers  
- Programming language preferences  
- Work-life balance & salary satisfaction metrics  
- Survey difficulty level analysis  

---

## 🛠 Tools Used

- Power BI  
- Excel (Data Cleaning)
